YITH Auctions for WooCommerce plugin allows buyers to grab products for their best price and vendors to generate profit, even bigger than expected.

== Description ==

YITH Auctions for WooCommerce gives you the possibility to create auction products in your shop. Users of your store will be able to bid on the desired items.


= Features: =
* Create unlimited auction products
* Assign a start date and end date to each auction product.
* Distinguish auction products by using a custom icon on both shop page and product page.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Auctions for WooCommerce` from Plugins page.

YITH Auctions for WooCommerce will add a new submenu called "Auctions" under "YITH Plugins" menu. Here you are able to configure all the plugin settings.
